Tools yang digunakan:
Matlab
Penulis menggunakan:matlab R2015b

Terdapat 2 file main yakni main1_quilting dan main2_quilting. 
Perbedaannya  pada main1 ukuran square bida sesuai keinginan user dengan memberikan inputan,
sedangkan pada main2 berapa ukuran input maka ukuran output juga sama

Cara menjalankan program Quilting : 
1. Buka file main1_quilting.m/main2_quilting pada matlab 
2. Tekan tombol Run pada matlab
3. Program akan berjalan, dan memberikan 3 pilihan yakni:
     square, notsquare horizontal dan not square vertical
4. Setelah memilih antara 3 pilihan tersebut selanjutnya memasukkan inputan ukuran di main1 apabila square, 
     dan memasukkkan banyaknya matrix yang diinginkan
5. Memilih warna yang diinginkan 
5. Program akan berjalan , dan menghasilkan motif baru
